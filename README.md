# logfactory
目前会再每天的早上2点自动执行，如果没有生成对应文件或者生成错误，可以手动执行，如下：
自动执行脚本：auto.sh
自动执行日志：auto.log

手动执行步骤：
1. node main/start
等全部执行完毕就大功告成了，去到与logfactory平级的hl目录下看生成的html文件吧



auto.sh为自动执行的脚本，与业务没有直接联系
